# Personal Portfolio Website

Repository ini berisi source code website **portofolio pribadi** yang digunakan untuk menampilkan profil singkat, keahlian, dan karya yang pernah dikerjakan.

## Tujuan
- Menjadi representasi profesional secara online  
- Menampilkan kemampuan dan pengalaman  
- Mendukung personal branding

## Konten
- Profil singkat
- Keahlian
- Proyek / pengalaman
- Informasi kontak

## Teknologi
- HTML
- CSS
- JavaScript

## Penggunaan
Buka `index.html` melalui browser dan sesuaikan konten sesuai kebutuhan.

---

Website ini dibuat sebagai portofolio pribadi.
